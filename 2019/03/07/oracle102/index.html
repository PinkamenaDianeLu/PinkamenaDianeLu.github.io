SELECT_01

Select语句是sql语句的重中之重，可以说日常的工作中使用最多的也就是select语句

也就是我们熟知的 select from表 where 条件 的意思可以理解为 所有

所以也可以写为

别名

当我们为了方便查看或是某些特定需求的时候，我们可以使用列别名来查询数据

更改显示的名字

可用于算符表达式

元名字加上空格 再加上别名就可以 也可以加上as

可以在别名上加上双引号 使其可以显示特殊符号，空格 和大小写(否则都是大写)

这样在查询结果处就会以你想要的列名出现

Concatenation operator：连接操作符

||
可以将两个东西连接起来或是在两个中间加上字面符号 用单引号引起来

简单的常用函数

在工作中经常会出现一些统计型的需求，比如查询最大值、最小值、总数之类，这时我们要使用一些oracle自带的函数来解决问题，这些函数并不是适用于所有数据库，但是在我们日常使用的oracle中非常实用

Count() 查询总数


Count()函数能够返回查询的总数目，用于统计总数，括号中推荐写1而不是写* 从而提高查询效率

Max()、Min() 查询最大值、最小值


Max()函数能够返回查询结果的列名的最大值,同理min()返回最小值

Length() 查询长度


可以用 select length(列名) from 表 来查询一个字段的长度，工作中常放在where后来筛选某字段长度，如统计2个字名字和三个字名字的人就可以用length()函数

upper() lower() 转换大小写



在查询结果为英文时，我们可以使用upper(列名) 来将结果转换为大写或是用lower(列名)将结果转换为小写，他也多用于where条件中，最常见的就是身份证号最后的x，在不同库中存储的大小写可能不同，那么就可以使用此函数将其统一

where

where不仅存在于select语句，他表示搜索的限定条件

Where 指明条件

Where子句是true false 还有no null (null在之后讲解)

Where子句中不能使用别名

常量的字符串要用单引号引起来

当涉及到字符串的比较时，区分大小写

查询时要符合格式(如日期要符合列中的对应格式)

可以使用 upper()或是lower()函数 将列的内容去哪不变成大写或小写 进行查询

关系运算符


只要where子句后的表达式为true时，即可返回相应数值，在计算机中，true与false往往表示真假、成立或不成立的意思，在sql中可以理解为 成立的条件

大于> 小于< 等于= 不等于>< 大于等于=> 小于等于=<

Select * from 表 where 列名 关系运算符  条件

and、or、not


这三个词就像我们所知的英语单词一样 and 和、or 或者、not 不是

AND 而且并且 两者都为true时

OR两者一个为true即可

Not 取反 如果是true 那么就是false

ORDER BY

使用order by 排序

ASC 升序排序(默认) DESC 降序排列

在整个select 句子中 order by 在句子最后，注意 order by 子句不是必须要在where后的

也可以按照列别名,列名顺序排序
也可以按多个列排序
(当出现null值 升序在最后 降序在最前 也可以用 nulls first 或是 nulls last 规定null值放在最前或最后)
